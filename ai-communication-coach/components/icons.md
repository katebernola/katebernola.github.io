# Icons

## What it is

The icon vocabulary: every icon has one defined purpose, used consistently everywhere. An icon is a message — it gets the same scrutiny as a sentence.

## Rules

- Every icon in the library has a stated purpose (the table below). Check the purpose before using an icon; never repurpose one — a magnifying glass that doesn't mean search breaks trust in every icon. *(source: Polaris icon library metadata)*
- Common actions (edit, delete, search, copy, add) have universally understood icons and need no text label. Ambiguous icons always get a label. *(source: Polaris using icons)*
- Once a meaning is assigned to an icon, use it consistently for that object or action everywhere. *(source: Polaris using icons)*
- Redundant icons are clutter: don't repeat the same icon down a table column. *(source: Polaris using icons)*
- Icons sit to the left of their label; when a label has multiple words, the icon represents the verb. *(source: Polaris common actions pattern)*
- Status severity: alert circle = warning, alert diamond = critical error, alert triangle = caution (see table). Pair with text — color and icon carry severity together, never color alone.
- Color meanings are cultural, not universal (red is failure in North America, prosperity in China). Icons chosen for global products use universal metaphors. *(source: Polaris internationalization)*

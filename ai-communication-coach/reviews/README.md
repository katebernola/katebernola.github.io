# Reviews

Every critique the coach generates gets saved here, dated, with enough context to stand alone. This folder is the training corpus the system grows from.

## File format

```markdown
# <Subject> review — <date>

## Context
What was reviewed, what stage, whose goal.

## Findings
Ranked, in the four-part frame (observation, reasoning, recommendation, tradeoffs).
Name any mistakes/ files that apply.

## Outcome
What was decided, if known. Update later if the results come in.
```

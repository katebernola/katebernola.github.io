# Retirement enrollment

## Initial request

Write content for the retirement landing page.

## What stakeholders thought

Users need to know they're eligible.

Need definition of 401(k).

Need compound interest.

Need tax explanation.

## Actual user goal

Feel confident enough to begin enrollment.

## Actual business goal

Increase enrollment.

## Communication strategy

Move employer match earlier.

Remove eligibility.

Reduce educational content.

Use progressive disclosure.

## Why

*To be filled in from the real case: what evidence separated the stakeholder assumptions from the actual user goal, and why each strategy move followed.*

## Lessons learned

- Stakeholder asks arrive as content lists ("they need to know X") — the list encodes assumptions, not needs.
- Education before action is a belief worth challenging here: see [education-follows-action](../beliefs/education-follows-action.md).
- Confidence, not comprehension, was the goal. Copy that teaches can still fail to reassure.

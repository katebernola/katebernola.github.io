# Product language

The product's own vocabulary: feature names, plan names, and the concepts the product owns.

*Empty on purpose — fill with real, approved names. Never guess a product name; a wrong one in shipped copy is a support ticket generator.*

## Rules

- Product names are the only exception to sentence case. Spell them exactly as branded, every time.
- Before using a capitalized name in copy, confirm it appears in this file. If it doesn't, either add it (with approval) or lowercase it.
- When a feature is renamed, append the old name here with a date — old names live on in tickets and search queries.

Key insight
Nothing here persuades. The comparison was always the argument — it was just hidden behind the choice it was supposed to inform.

Reuses the two-column comparison already built for the auto-file accept/decline screen. Neither option is pre-selected: the paid option benefits the business, so pre-choosing it is a deceptive pattern. The popularity badge is only usable while it stays true, with an owner and a review date.

Artifact 2 — message states

The dashboard alert is the loudest cue in the product, and today it clears when the ID is added — while the user is still two steps from being able to have taxes filed. The fix isn't to make it persistent. It's to make it accurate.
